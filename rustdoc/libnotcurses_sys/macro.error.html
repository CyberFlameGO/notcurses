[−][src]Macro libnotcurses_sys::error

macro_rules! error {
    ($res:expr, $msg:expr, $ok:expr) => { ... };
    ($res:expr, $msg:expr) => { ... };
    ($res:expr) => { ... };
}

Returns an Ok($ok), or an Err(NcError) if $res < NCRESULT_OK.

In other words: Returns Ok($ok) if $res >= NCRESULT_OK, otherwise returns Err(NcError::new($res, $msg)).

$ok & $msg are optional. By default they will be the unit type (), and an empty &str "", respectively.
