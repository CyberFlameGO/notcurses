[−][src]Type Definition libnotcurses_sys::Notcurses

type Notcurses = notcurses;

Notcurses builds atop the terminfo abstraction layer to provide reasonably portable vivid character displays.

This is the internal type safely wrapped by FullMode.

pub fn getc(
    &mut self,
    time: Option<NcTime>,
    sigmask: Option<&mut NcSignalSet>,
    input: Option<&mut NcInput>
) -> NcResult<char>[src]

Returns a char representing a single unicode point.

If an event is processed, the return value is the id field from that event.

Provide a None time to block at length, a time of 0 for non-blocking operation, and otherwise a timespec to bound blocking.

Signals in sigmask (less several we handle internally) will be atomically masked and unmasked per ppoll(2).

*sigmask should generally contain all signals.

C style function: notcurses_getc().
